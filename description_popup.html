Global Shortcuts: Want keyboard shortcuts to work even when Chrome is minimized?
Change the shortcuts settings to "Global"!

Setup Global Shortcuts:

1

Go to chrome://extensions/shortcuts

2
Set all the extension's shortcuts to 'Global'.
3
Restart Chrome (about://restart) if this doesn't work right away
4
Now you can control playback even if Chrome browser is minimized!

Customize shortcuts

5
You can customize the shortcuts as you like:
Go to chrome://extensions/shortcuts and modify the default shortcuts as you please.
For example, Ctrl-J, Ctrl-K, Ctrl-L, Alt-A, Alt-S etc...

Note that the 'space' key can't be used when setting custom shortcuts.
